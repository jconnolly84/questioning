Class Voting

A quick, low-stakes way to get a response from every pupil. Use it for retrieval, misconceptions, and to spark debate: ‘convince me your choice is right’.

Video

Watch the demonstration, then use the notes and resources to plan how you’ll run it tomorrow.

How to run it (quick routine)

  • Set the rule: everyone thinks first (no hands up unless invited).
  • Ask one clear question; pause long enough for real thinking.
  • Collect responses from across the room (not just keen hands).
  • Probe: “How do you know?” “What makes you say that?” “Can anyone improve it?”

Why it works

Class voting gets a response from everyone in seconds. It’s brilliant for retrieval, misconceptions, and decisions (“Which method is best?”). The magic is what you do next: ask pupils to justify and challenge each other.

  • High participation with low friction.
  • Reveals misconceptions quickly.
  • Creates purposeful talk: justify, compare, persuade.

Run it like a pro

  • Use A/B/C/D or thumbs or hands — keep it consistent.
  • Don’t accept “because I think so” — demand a reason.
  • Ask: “Who chose a different option? Convince us.”
  • Re-vote after the discussion to show learning happening.

Great question types

  • Which example best fits the definition?
  • Which step comes next in the process?
  • Which answer includes the misconception — and why?
