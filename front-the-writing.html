TECHNIQUE • FRONT THE WRITING

Front the Writing

Get pupils to write an answer first, then improve it through discussion, examples, and success criteria. It boosts quality and reduces ‘I don’t know’.

Video

Watch the demonstration, then use the notes and resources to plan how you’ll run it tomorrow.

How to run it (quick routine)

  • Set the rule: everyone thinks first (no hands up unless invited).
  • Ask one clear question; pause long enough for real thinking.
  • Collect responses from across the room (not just keen hands).
  • Probe: “How do you know?” “What makes you say that?” “Can anyone improve it?”

Why it works

Pupils often “know it” but can’t express it. Front the Writing makes the thinking visible first, then improves it using modelling, vocabulary and success criteria. The result is better extended responses and fewer blank pages.

  • Gets something on the page (reduces ‘I don’t know’).
  • Makes improvement concrete (edit, upgrade, polish).
  • Builds exam-style habits: plan → write → refine.

Upgrade prompts

  • “Add because…” (reasoning)
  • “Add an example…”
  • “Use a key term…”
  • “Make it more precise…”

Make it stick

Repeat the same cycle every lesson: short write, share a great model, then 60 seconds of red-pen improvement. That repetition is what turns good writing into a habit.
