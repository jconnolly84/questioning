TECHNIQUE • FRONT THE WRITING

Front the Writing

Get pupils to write an answer first, then improve it through discussion, examples, and success criteria. It boosts quality and reduces ‘I don’t know’.

Video

Watch the demonstration, then use the notes and resources to plan how you’ll run it tomorrow.

How to run it (quick routine)

  • Set the rule: everyone thinks first (no hands up unless invited).
  • Ask one clear question; pause long enough for real thinking.
  • Collect responses from across the room (not just keen hands).
  • Probe: “How do you know?” “What makes you say that?” “Can anyone improve it?”
