QUESTIONING • OVERVIEW

Better questions. Better thinking. Better learning.

This site is a practical, classroom-ready guide to high-impact questioning routines. Use it to sharpen thinking, increase participation, and check understanding in real time — without turning lessons into a Q\&A with the same few hands up.

How to run it (quick routine)

  • Set the rule: everyone thinks first (no hands up unless invited).
  • Ask one clear question; pause long enough for real thinking.
  • Collect responses from across the room (not just keen hands).
  • Probe: “How do you know?” “What makes you say that?” “Can anyone improve it?”

What great questioning does

Questioning isn’t a performance. It’s your diagnostic tool — the fastest way to find out what pupils know, what they think they know, and what they’ve misunderstood. The aim is simple: get every pupil thinking, then use what you hear (and see) to decide what happens next.

  • Participation: widen it (not just the keen hands).
  • Thinking time: build it in so answers improve.
  • Follow-up: ask “why?” and “how do you know?” to deepen understanding.
  • Response systems: use routines (whiteboards/voting) to check everyone quickly.

The 3 golden rules

  • Ask one clear question (no double-barrelled waffle).
  • Pause longer than feels comfortable — that’s where thinking happens.
  • Probe and improve: “Can you add evidence?”, “Can you make it more precise?”, “Who can build on that?”

Pick the right routine

Use the technique pages like a toolkit. If you want every pupil, go to Mini Whiteboards or Class Voting. If you want better verbal answers, go to Pose–Pause–Pounce, Think–Pair–Share or Cold Calling. If you want a decision point, go to Hinge Questions.
